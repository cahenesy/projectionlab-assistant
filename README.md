# ProjectionLab Assistant

A Chrome extension that adds an AI-powered chatbot sidebar to [ProjectionLab](https://ww.projectionlab.com) for natural-language financial plan analysis and automated configuration updates.

**Repository:** https://github.com/cahenesy/ProjectionLabAssistant

## Features

- Sidebar chatbot interface in ProjectionLab
- Connects to any popular LLMs (configurable)
- Analyzes your plan data via `exportData()`
- Parses natural language to apply updates via ProjectionLab Plugin API
